## 📋 接口分类

### 🔐 用户认证

- 用户注册、登录、获取用户信息

### 🎵 文本转语音 (TTS)

- 语音生成、生成历史记录

### 🔒 双因素认证 (TOTP)

- TOTP 设置、验证、备用恢复码

### 🔑 Passkey 认证

- Passkey 注册、认证流程

### 👨‍💼 管理员功能

- 用户管理、系统管理

### 🤖 LibreChat 集成

- 聊天功能、历史记录

### 📊 其他功能

- 数据收集、日志管理、安全防护
